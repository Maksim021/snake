Создана функция Input для обработки пользовательского ввода для управления игрой.

Параметры

- Параметры: отсутствуют.

Алгоритм работы

1. Проверка нажатий клавиш:
   - Используется _kbhit() для проверки наличия нажатой клавиши.

2. Обработка ввода:
   - С помощью _getch() считывается нажатая клавиша:
     - 'a' - направление LEFT.
     - 'd' - направление RIGHT.
     - 'w' - направление UP.
     - 's' - направление DOWN.
     - 'x' - завершение игры (установит gameOver в true).

Использование: 

Функция вызывается в игровом цикле для обновления направления змеи и управления завершением игры.
